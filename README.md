## Qu'est-ce que Biblialgo ?
Il s'agit d'une collection d'algorithmes en C# qui sont utiles dans bon nombre de programmes<br>
Les tests sont également disponibles sur le projet afin que vous puissiez modifier ou bien tester vos propres programmes.

## Comment a-t-il été réalisé ?
Ce projet a été réalisé en C#. Les tests ont été fournis pas l'équipe pédagogique de l'UFR ST.

## Guide d'installation

#### Etape 1
Clonez le projet sur votre orinateur.

#### Etape 2
Rendez vous dans le dossier

#### Etape 3
Pour lancer les tests effectuez la commande ./run_tests

#### Etape 4
Pour modifier les fichiers contenants les fonctions, rendez vous dans le dossier ./lib
